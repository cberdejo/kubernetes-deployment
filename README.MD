# Kubernetes Learning Phases

This section of the repository gathers my hands-on Kubernetes learning path in a practical, progressive way.  
The goal is to make each phase useful both for my own review and for anyone who wants to learn by following real examples.

The project is still a work in progress, and new phases will be added over time.

## Related Documentation

- Roadmap: `docs/roadmap.md` - full long-term learning plan and upcoming stages.
- Glossary: `docs/glossary.md` - key concepts and terminology used across the phases.

## Implemented Phases

| Phase | Folder | What you will do |
| --- | --- | --- |
| Phase 00 | `phases/00-local-compose` | Run the full app locally with Docker Compose, understand container networking, environment variables, and persistence basics. |
| Phase 01 | `phases/01-kubernetes-basics` | Learn Kubernetes fundamentals by working with Pods, Deployments, Services, and basic cluster operations with `kubectl`. |
